# 🔐 Blockchain Email Authentication System

Дипломный проект — система аутентификации email-сообщений
с использованием блокчейна, RSA криптографии и SHA-256.

## Технологии
- Python + Flask
- RSA-2048 + SHA-256
- Web3.py + Ganache
- SQLite
- Vanilla JS

## Запуск

### 1. Установи зависимости
pip install flask flask-cors cryptography colorlog

### 2. Запусти Ganache
ganache --port 7545 --chain.chainId 1337

### 3. Запусти приложение
python main.py

### 4. Открой браузер
http://localhost:5000
